Change your browser tab title to catch users' attention even when they open multiple tabs. Easily configurable through the plugin settings.

== Description ==

Attention-Grabbing Tab Title is a plugin designed to help you catch your users' attention by animating the browser tab title when they leave your site. The plugin allows you to:

- Enable or disable the feature.
- Add multiple titles that will be displayed in an animated sequence.
- Specify the duration in seconds for each title change.
- Optionally, restrict the feature to only the homepage.

When enabled, this plugin ensures that your e-commerce site remains noticeable even with many tabs open.

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload `Attention Browser Tab Title` to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Navigate to the plugin settings page in the WordPress admin menu to configure the options.

== Frequently Asked Questions ==

= How do I configure the plugin? =

Go to the plugin settings page under the 'Tab Title Settings' menu item. There, you can enable the feature, add title texts, set the title change interval, and choose if the feature should be enabled only on the homepage.

= Can I add multiple titles? =

Yes, you can add multiple titles. Use the 'Add Title' button to include more titles, and you can remove any unnecessary titles using the 'Remove' button next to each title field.
